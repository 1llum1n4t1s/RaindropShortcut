# プライバシーポリシー - Raindrop Shortcut

最終更新日: 2026年4月18日

## はじめに

「Raindrop Shortcut」（以下「本拡張機能」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。本プライバシーポリシーは、本拡張機能におけるデータの取り扱いについて説明します。

## 収集するデータ

本拡張機能は、個人情報を一切収集しません。

## ローカルに保存するデータ

本拡張機能は、以下の設定データをユーザーの端末内（`chrome.storage.local`）にのみ保存します。

- OAuth アクセストークン・リフレッシュトークン（Raindrop.io との認証用）
- 選択中のコレクション情報
- テーマ設定（自動 / ライト / ダーク）
- リンクの開き方設定（新しいタブ / 現在のタブ）
- ブックマーク一覧のローカルキャッシュ（popup 再表示の高速化目的、TTL 5 分）

これらのデータは端末内にのみ保存され、Raindrop.io 以外の外部サーバーへの送信は一切行いません。

## データの共有

本拡張機能は、いかなるデータも第三者と共有しません。

## ネットワーク通信

本拡張機能は、以下の目的でのみ外部通信を行います。

- **raindrop.io** — OAuth 2.0 認証（ログイン・トークン更新）
- **api.raindrop.io** — ブックマーク・コレクション情報の取得
- **www.google.com (Favicon Service)** — ブックマーク一覧に表示する favicon 画像の取得のため、各ブックマークのドメイン名のみを送信します。ブックマーク自体のタイトル・URL・その他の個人識別情報は送信しません。

上記以外のサーバーとの通信は一切行いません。

## 権限の使用目的

- **identity**: Raindrop.io の OAuth 2.0 認証フローを実行するために使用します。
- **storage**: 認証トークン、コレクション選択、テーマ設定、リンク設定、ブックマークキャッシュをローカルに保存・復元するために使用します。
- **host_permissions (api.raindrop.io, raindrop.io)**: Raindrop.io の公式 API からブックマーク・コレクション情報を取得し、OAuth 認証のトークン交換を行うために使用します。

## 変更について

本プライバシーポリシーは予告なく変更される場合があります。変更があった場合は、本ページを更新します。
